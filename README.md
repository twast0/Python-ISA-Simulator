# Python-ISA-Simulator
ISA Simulator in Python


Usage:

python isa-sim.py <amount_of_cycles> <program_file.txt> <initial_data_memory.txt>

Sample test:

Figures out if the number stored on address 0 in the data memory (data_mem_3.txt) is a prime number.

Limited to 8-bits and will wrap-around if value is outside of this range.

If the number is prime, the output will show a 1 at address 10 of the data memory.
